# Screenshots

`*-v060.png` 是 Agentic-Island `0.6.0` 的真实 Electron 界面截图。

这些图片由以下命令生成：

```powershell
npm run build
npm run docs:capture
```

截图脚本使用临时 Electron `userData` 和仓库内定义的演示数据，结束后删除临时配置。不得使用个人配置目录直接生成公开截图。

|---|---|
| `ask-v060.png` | 问答（会话分支、上下文、多模型协作） |
| `shortcuts-v060.png` | 快捷 |
| `todos-v060.png` | 待办 |
| `notes-v060.png` | 灵感便签 |

| `settings-v060.png` | 设置 |
