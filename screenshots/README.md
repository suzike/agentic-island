# Screenshots

`*-v063.png` 是 Agentic-Island `0.6.3` 的真实 Electron 界面截图。

这些图片由以下命令生成：

```powershell
npm run build
npm run docs:capture
```

截图脚本使用临时 Electron `userData` 和仓库内定义的演示数据，结束后删除临时配置。不得使用个人配置目录直接生成公开截图。
