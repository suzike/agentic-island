# Agentic-Island v0.6.4

发布日期：2026-07-22

`v0.6.4` 将终端从单次运行的 ConPTY 面板升级为可恢复的开发工作区，并修复终端获得键盘焦点后仍可能被灵动岛自动收起动画带动的问题。问答、模型、项目工作台、待办、便签、资讯、截图和录屏等既有能力保持不变。

## 主要变化

- 标签、工作目录、Shell、历史、退出码、耗时、项目工作区、启动任务和环境配置写入版本化终端现场；Windows DPAPI 可用时整体加密。
- 启动恢复中心支持选择会话、选择启动任务、自动恢复或进入空白现场；只创建新 PTY，不伪造已经结束的进程。
- 增加 PowerShell 5.1、PowerShell 7、CMD 与 WSL 配置，支持加密环境变量配置和安全工作区导入导出。
- 增加项目任务扫描、Git 上下文、持久历史与收藏、输出折叠摘要、路径拖入/点击、危险命令二次确认。
- 增加基于当前目录、最近命令和有限输出的 AI 诊断、交接摘要与下一步规划，建议命令不会自动执行。
- 输出快照默认关闭；开启后执行自动脱敏、保留期和容量限制，导出不包含输出与环境变量值。
- 终端键盘焦点会锁定灵动岛展开状态，延迟收起回调执行前再次检查焦点，避免输入时整个终端框上移。
- PowerShell 5.1 提示符控制序列改用兼容的 `[char]27` / `[char]7`，不再暴露 OSC 标记文本。

## 已知边界

- 当前只支持 Windows 10/11 x64，终端依赖 ConPTY。
- 安装包未做商业代码签名，Windows SmartScreen 可能显示未知发布者。请仅从本仓库 GitHub Releases 下载并核对 SHA-256。
- 云模型、Embedding、GitHub 私有数据和 CalDAV 能力仍需用户自行提供相应服务配置。
