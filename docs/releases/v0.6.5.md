# Agentic-Island v0.6.5

发布日期：2026-07-23

`v0.6.5` 是终端多会话输入稳定性补丁。它修复第一个 PowerShell 会话正常，但新建第二个会话后输入框再次抖动的问题。既有问答、模型、项目工作台、待办、便签、资讯、截图和录屏能力保持不变。

## 修复

- PowerShell 5.1/7 的 UTF-8 编码和退出码 OSC 提示符初始化改为 `-NoExit -Command` 启动参数，在进入交互模式前完成。
- 不再在 PTY 创建并获得 xterm 焦点后写入长初始化命令，消除第二个终端启动时与用户输入交错的竞争条件。
- 新建、切换标签和 xterm 挂载会在焦点迁移前取消待执行的自动收起计时器，避免主面板在第二终端输入期间向上滑走；手动 Esc 收起不会因此重新弹出。
- CMD 与 WSL 会话启动参数保持原样。

## 安装

安装包：`Agentic-Island-Setup-0.6.5.exe`

Windows 10/11 x64。可覆盖安装旧版本，用户配置与终端工作区保留在 Electron `userData`。

## 已知限制

- 当前只支持 Windows 10/11 x64，终端依赖 ConPTY。
- CMD 与 WSL 不支持 PowerShell OSC 退出码标记时，状态栏退出码保持未知。
