# Agentic-Island v0.6.0

`v0.6.0` 是问答会话能力、模型供应商配置和动态灵感体验的完整升级版本。既有 Agent、快捷、待办、便签、资讯、复盘、终端、截图和录屏能力全部保留。

## 主要更新

- 从任意提问或回答 Fork 会话，保留父子谱系、独立命名、分支切换和 AI 压缩合并。
- 分支级长期记忆与持续指令真实注入模型；支持上下文钉选、排除、附件连续上下文和 token 预算检查。
- 从已保存模型中选择 2–4 个并行作答，支持并列比较、共识提炼、分歧辩论和采用候选回答。
- 新增回答自检、假设与风险、替代路径、问题拆解、苏格拉底式澄清、知识库核验和下一问建议。
- 完整分支、单条回答或框选片段可直接写入本地向量知识库，立即参与后续 RAG 检索。
- DeepSeek、Kimi 等供应商分别保存模型、Base URL 和 API Key；支持旧配置迁移、在线模型同步和认证错误脱敏。
- 问答空态推荐与常驻迷你条共用动态内容池和轮播节奏，持续刷新 AI、GitHub 与工作简报内容。

## 兼容与数据

- 旧版 `askThread` 会自动迁移为根分支，不删除已有问答记录。
- 当前分支、归档分支、长期记忆和持续指令随应用状态加密持久化。
- 原有问答、引用追问、气泡内追问、云模型、本地 Claude Code/Codex 和知识库模式继续可用。
- 供应商配置迁移保留已有自定义模型，并阻止不同供应商之间串用密钥。
- 审计、截图和安装验证实例可使用独立 bridge discovery 文件，不覆盖真实 Agent 通信通道。
