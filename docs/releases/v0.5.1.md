# Agentic-Island v0.5.1

`v0.5.1` 是专业录屏工坊的可靠性与发布同步版本。

## 主要更新

- 修复显示器选择、窗口来源标签和应用自身窗口过滤问题。
- 区域、画面比例、铺满/完整显示、动态运镜和定位框统一使用真实源裁剪坐标。
- 窗口来源在媒体轨就绪后使用实际视频尺寸，避免估算定位偏差。
- 修复暂停恢复时间线、收起控制条导致 Canvas 卸载、关键帧重复/丢失和健康度误报。
- 补全录屏实时取景小地图、分片落盘、工程恢复和 FFmpeg 多格式导出的验证。
- 全量同步 README、架构图、录屏管线图、开发指南、发布指南和当前版本截图。

## 验证

- 两套 TypeScript 配置检查通过。
- 31 个离线测试脚本通过。
- Electron main/preload/renderer 生产构建通过。
- 定位框与实际源裁剪共用坐标映射，录屏几何回归测试通过。
- NSIS x64 安装包构建与隔离安装启动验证通过。
- `npm audit --omit=dev`：生产依赖漏洞为 0。
