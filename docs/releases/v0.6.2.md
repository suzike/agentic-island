# Agentic-Island v0.6.2

发布日期：2026-07-21

## 本版重点

- 修复问答页与设置页模型切换不一致：供应商、型号、Base URL 和 API Key 原子切换，同模型不同账号可独立保存、识别和参与多模型讨论。
- 每条新云端回答和气泡内追问记录实际请求模型，切换顶部模型后仍能确认历史回答归属。
- OpenAI 内置目录更新为 GPT-5.6 Sol / Terra / Luna，并按快速/深度模式发送正确的完成长度与 reasoning effort 参数。
- Claude 内置目录更新为 Sonnet 5、Opus 4.8、Fable 5、Haiku 4.5，并按型号能力映射 adaptive thinking、effort、思考摘要和手动预算。
- 知识库新增独立 Embedding Base URL、模型和 API Key。旧配置自动迁移一次，之后切换聊天模型不再破坏索引、语义检索或会话写入。
- 保留原有每条回答气泡下的“继续追问”支线；分支、上下文、回答分析和会诊结果的交互结构未被取消。

## 兼容与迁移

- 旧 Claude Sonnet 4 / 4.6 配置迁移到 Sonnet 5；旧 Opus 4 / 4.6 / 4.7 配置迁移到 Opus 4.8。
- 旧版仅保存 `embedModel` 时，会在首次启动复制当时的聊天 Base URL 与 API Key 形成独立向量连接；用户之后可在知识库面板单独修改或清空。
- Kimi Code 继续使用 `https://api.kimi.com/coding/v1` 与 `kimi-for-coding`、`kimi-for-coding-highspeed`、`k3`；Kimi 开放平台继续使用 `https://api.moonshot.cn/v1`，两类密钥互不通用。

## 验证

- TypeScript main/preload/shared 与 renderer/shared 两套检查通过。
- 33 个离线测试脚本全部通过，包含气泡内追问、分支回写、供应商/账号迁移、GPT-5.6、Claude 5、Kimi、DeepSeek 和独立 Embedding 回归。
- Electron main、preload、renderer 三端生产构建通过；生产依赖审计为 0 个已知漏洞。
- 8 张 README 截图由隔离 Electron 实例重新采集并人工检查问答与设置页面。
- `win-unpacked` 启动、NSIS 静默安装、安装后启动、隔离 bridge 和静默卸载验证通过。

真实付费 API 连通性未纳入自动化测试，发布验证不使用或上传个人密钥；请求体与模型目录以各供应商官方文档和本地协议回归为准。
