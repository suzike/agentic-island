# Agentic-Island v0.2.0

`v0.2.0` 将 Agentic-Island 从顶部 Agent 审批面板扩展为围绕项目上下文运行的本地工程工作台。

## 主要更新

- 11 个主分区与全局命令工具。
- 资讯 → 待办 → 快捷执行 → 成果 → 复盘的项目闭环。
- 本地 Claude/Codex Agent、知识库 RAG、第二大脑和截图视觉工作流。
- 快捷、待办、便签、Markdown、资讯、终端和复盘的完整增强。
- 外部网页、文件、文件夹、会议和文件对话框统一让位，不再被灵动岛覆盖。
- Electron 39、统一离线测试入口、隔离文档截图与全量架构文档。

## 安装

下载 `Agentic-Island-Setup-0.2.0.exe`。当前安装包未使用商业代码签名，Windows SmartScreen 可能提示未知发布者。
