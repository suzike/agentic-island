# Agentic-Island v0.6.3

发布日期：2026-07-22

`v0.6.3` 是终端交互可靠性补丁版本，集中修复 PowerShell/交互式 Agent CLI 输入卡顿、提示行抖动和工作目录无法通过系统对话框选择的问题。问答、模型、项目工作台、待办、便签、资讯、截图和录屏等既有能力保持不变。

## 主要变化

- PTY 输出只在 PowerShell 提示符目录真实变化时更新 React 状态；重复识别到相同目录会保持原数组引用。
- xterm fit 的连续 `ResizeObserver` 通知合并到同一绘制帧，移除渲染层重复 resize；主进程按会话记录最后 `cols/rows`，相同尺寸不再调用 ConPTY。
- PowerShell 工具栏的文件夹入口改为 Windows 原生目录选择器，选中后立即执行安全转义的 `Set-Location -LiteralPath`。
- 目录选择器接入 External Yield 原生对话框包装，选择期间灵动岛暂时释放最高窗口层级。
- 补充终端状态引用稳定性和原生目录选择器回归，并同步 README、架构、开发、发布、截图和历史设计状态说明。
